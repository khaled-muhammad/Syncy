What is Syncy?

Syncy lets you sync and watch any media file with friends in real-time across devices ‚Äî your private virtual theater with AI-powered search, seamless sync, and beautiful UI.

Key Features

Smart Media Discovery

Auto-scans your device, AI-powered fuzzy search, thumbnail previews.

Real-Time Sync

WebSocket-backed sync with sub-second latency and perfect playback.

Glassmorphic UI

Blur effects, animations, and gesture navigation built for binge-watching.

Smart Search Engine

Fuzzy matching, typo handling, and relevance ranking done right.

Tech Stack

Frontend: Flutter 3.8, GetX, Realm, Custom Animations

Backend: WebSockets, Dio HTTP, File system integration

Advanced: Isolates, Smart Caching, Permission Handling

Getting Started

Clone & Run:

$ git clone https://github.com/khaled-muhammad/Syncy.git
$ cd Syncy
$ flutter pub get
$ dart run realm generate
$ flutter run
